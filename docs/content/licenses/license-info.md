## Third party

The [License Maven Plugin](https://www.mojohaus.org/license-maven-plugin) is used to extract all license information out of the pom.xml (all modules). The list can be found [here](THIRD-PARTY.md)



To re-generate the license list (in root folder):

```
mvn clean verify
```
