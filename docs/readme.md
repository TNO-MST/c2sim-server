# MKDoc

https://www.mkdocs.org/

## Setup

```
# 1. Create a Virtual Environment
python -m venv .venv
# 2. Activate the Environment
.venv\Scripts\activate
# 3. Install MkDocs
pip install mkdocs
pip install mkdocs-material
```

## Run server

```
mkdocs serve -a 0.0.0.0:7778
```
